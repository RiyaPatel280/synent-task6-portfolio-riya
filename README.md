#  GitHub Profile Finder

A modern and responsive web application that fetches and displays GitHub user profile data using the GitHub API.  
This project allows users to search for any GitHub profile and view key details in a clean UI.

---

##  Features

- Search GitHub users by username
- Display user profile information
- Show avatar, name, and bio
- Display followers, following, and repository count
- Direct link to GitHub profile
- Loading state and error handling
- Clean and modern UI design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Fetch API, DOM Manipulation)

---

##  How It Works

- User enters a GitHub username
- Application sends a request to the GitHub API
- Data is fetched using **Fetch API (async/await)**
- Profile information is dynamically rendered on UI
- Error message is shown if user is not found
